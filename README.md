## 🏥 Descripción

Sistema de gestión hospitalaria diseñado para clínicas y hospitales pequeños.  
Permite administrar pacientes, usuarios, citas, historial médico y procesos internos de manera eficiente mediante una aplicación web moderna.

## 👥 Roles del Sistema

### 🔧 Administrador
- Gestión completa del sistema  
- Crear, editar y eliminar usuarios  
- Gestionar pacientes  
- Crear calendarios de asistencia  
- Registrar asistencia  
- Generar reportes mensuales  

---

### 🧑‍💼 Recepcionista
- Registrar nuevos pacientes  
- Editar información de pacientes  
- Generar turnos (tokens) para atención  

---

### 👨‍⚕️ Doctor
- Registrar historial clínico  
- Registrar síntomas y diagnósticos  
- Prescribir medicamentos  
- Solicitar pruebas de laboratorio  
- Guardar resultados clínicos  

---

## 🚀 Características Principales

- Sistema multiusuario  
- Interfaz moderna con Vue.js  
- Gestión completa de pacientes  
- Control de citas y turnos  
- Registro de historial médico  
- Reportes administrativos
